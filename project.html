Miami University of Ohio Virtual Tour

Responsive Web Site

The Miami University of Ohio virtual tour is a responsive web site dedicated to showing off the different facets of the campus, students, and faculty.

The virtual tour was created with HTML, CSS, and JavaScript. It was created as a responsive web site because the client wanted to make sure that it could be accessed on a wide range of devices. This decision was validated when post-launch analytics showed close to 20 percent of users browsing the site using a mobile device.
